MyFlix API Documentation

Movies

Get All Movies

Endpoint: GET /movies

This endpoint retrieves a list of all movies.

Response:

    [
      {
        "_id": "movie_id",
        "Title": "Movie Title",
        "Description": "Movie Description",
        "Genre": {
          "Name": "Genre Name",
          "Description": "Genre Description"
        },
        "Director": {
          "Name": "Director Name",
          "Description": "Director Description"
        },
        "ImageUrl": "movie_image.jpg",
        "Featured": true
      }
    ]

Get Movie by Title

Endpoint: GET /movies/:title

This endpoint retrieves data about a single movie by its title.

Response:

    {
      "_id": "movie_id",
      "Title": "Movie Title",
      "Description": "Movie Description",
      "Genre": {
        "Name": "Genre Name",
        "Description": "Genre Description"
      },
      "Director": {
        "Name": "Director Name",
        "Description": "Director Description"
      },
      "ImageUrl": "movie_image.jpg",
      "Featured": true
    }

Users

Register a New User

Endpoint: POST /register

This endpoint allows users to register by providing their information in the request body.

Request Body:

    {
      "Username": "user123",
      "Password": "password123",
      "Email": "user@example.com",
      "Birthday": "1990-01-01"
    }

Response:

    {
      "_id": "user_id",
      "Username": "user123",
      "Email": "user@example.com",
      "Birthday": "1990-01-01",
      "FavoriteMovies": [],
      "__v": 0
    }

Update User Information

Endpoint: PUT /users/:Username

This endpoint allows users to update their information. Provide the updated user data in the request body.

Request Body:

    {
      "Username": "new_username",
      "Password": "new_password",
      "Email": "new_email@example.com",
      "Birthday": "1995-02-15"
    }

Response:

    {
      "_id": "user_id",
      "Username": "new_username",
      "Email": "new_email@example.com",
      "Birthday": "1995-02-15",
      "FavoriteMovies": [],
      "__v": 0
    }

Add Movie to User's Favorites

Endpoint: POST /users/:Username/movies/:MovieID

This endpoint allows users to add a movie to their list of favorite movies.

Response:

    {
      "_id": "user_id",
      "Username": "user123",
      "Email": "user@example.com",
      "Birthday": "1990-01-01",
      "FavoriteMovies": ["movie_id"],
      "__v": 0
    }

Remove Movie from User's Favorites

Endpoint: DELETE /users/:Username/movies/:MovieID

This endpoint allows users to remove a movie from their list of favorite movies.

Response:

    {
      "_id": "user_id",
      "Username": "user123",
      "Email": "user@example.com",
      "Birthday": "1990-01-01",
      "FavoriteMovies": [],
      "__v": 0
    }

Deregister User

Endpoint: DELETE /users/:Username

This endpoint allows users to deregister. No request body is required.

Response:

"user123 deleted."
